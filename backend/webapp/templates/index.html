This is a small website where you can find events in FOSDEM 2025 based on content search, and via connections.

I (mike moran) wrote this because, whilst I really enjoy attending FOSDEM, the amount of possible things to see is large, and there is always a chance I can miss something.

The main website does have search, so, to be totally honest I should also say initially I just wanted to play about with OpenAI Embeddings. However, one outcome is that you'll find results for queries here you won't find on main site e.g. "controversial".

Additionally, this site also indexes related content, like slides (I'm not 100% sure, but I don't think the main site does that).

See Github for how this was built, and also for any pending / done TODO's.
